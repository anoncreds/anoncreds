# Code of Conduct

Hyperledger AnonCreds requires adherence to the [LF Decentralized Trust Code of Conduct](https://lf-decentralized-trust.github.io/governance/governing-documents/code-of-conduct/). All Working Group contributors and meeting attendees are asked
to review and follow the code of conduct.
